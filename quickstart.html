SlowMoMan ("Slow Motions on Manifolds") is a tool for exploratory data analysis. The fastest way to understanding is to do, so we present a simple example for users to run through.

Before using SlowMoMan, we need 2 items:
1. A dataset with many features (columns)
2. A 2D embedding of the above dataset
To obtain the 2D embedding, users can choose any embedding algorithm they wish (PCA, UMAP, t-SNE).

In this example, we start use data obtained by Zeisel et al. (2015). The original dataset has 2180 features, each of them a gene. There are 2816 rows, each corresponding to a single cell from the mouse brain. Each of the cells in this dataset belong to 1 of 7 subclasses of brain cell (interneurons, pyramidal SS, pyramidal CA1, oligodendrocytes, microglia, endothelial-mural, astrocytes_ependymal). From this large dataset, we run t-SNE to obtain a 2D embedding of the original data. Finally, we have all we need to being using SlowMoMan.
